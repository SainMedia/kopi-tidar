KOPI TIDAR PWA

Ini versi PWA/installable. Agar tombol Install muncul dan data lebih aman, file harus dibuka lewat hosting HTTPS, bukan langsung dari file manager.

Cara paling gampang:
1. Upload semua isi folder ini ke Netlify Drop / GitHub Pages / Vercel.
2. Buka link website dari Chrome HP.
3. Klik titik tiga Chrome > Add to Home screen / Install app.
4. Setelah ter-install, buka dari ikon Kopi Tidar di layar utama.

Catatan penting:
- Kalau dibuka langsung sebagai file:// dari File Manager, browser HP sering memperlakukan data sebagai sementara, jadi data bisa hilang.
- Versi ini tetap menyimpan data lokal di device, tapi ada menu Backup untuk export JSON dan Restore JSON.
- Kalau ingin data sinkron antar HP, perlu database online seperti Supabase/Firebase.
